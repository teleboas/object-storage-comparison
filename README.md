# Object Storage Pricing Comparison

Interactive calculator comparing cloud object storage pricing across major providers.

**Live demo:** https://teleboas.github.io/object-storage-comparison/

## Features

- Compare 8 cloud storage providers: AWS S3, Azure Blob, Google Cloud Storage, DigitalOcean Spaces, OVH, Hetzner, Scaleway, Backblaze B2
- Tiered pricing support for AWS, Azure, and GCP
- Free egress/storage tier handling
- EUR/USD display with unified sorting
- Responsive design
